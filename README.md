# Elaine Confeitaria Frontend

Frontend administrativo em `Next.js`, `React`, `TypeScript` e `MUI`, consumindo a API da collection Postman do backend.

## Stack

- `Next.js` com App Router
- `TypeScript`
- `MUI`
- `React Hook Form` + `Zod`
- `TanStack Query`
- `Axios`
- `Vitest` + Testing Library

## Rotas implementadas

- `/login`
- `/home`
- `/pedidos`
- `/carrinho`
- `/catalogo`

## Configuracao

1. Copie `.env.example` para `.env.local`
2. Ajuste `NEXT_PUBLIC_API_URL`
